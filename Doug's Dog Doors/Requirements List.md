# 🐶 Todd and Gina’s Dog Door (Requirements List – v2.3)

---

## 📋 System Requirements

### 1. Door Size Requirement
- The dog door opening must be at least **12 inches tall**.

---

### 2. Remote Control Operation
- A button on the remote control:
  - Opens the dog door if it is closed  
  - Closes the dog door if it is open  

---

### 3. Automatic Closing
- Once the dog door has opened, it should **close automatically** if it is not already closed.

---

### 4. Bark Recognition
- A bark recognizer must be able to detect when a dog is barking.

---

### 5. Automatic Door Opening via Bark
- The bark recognizer must open the dog door when it detects barking.

---

## 🧠 Notes

- Requirements **4 and 5 introduce automation** using bark detection.
- The system now supports **two input methods**:
  - Remote control (manual)
  - Bark recognizer (automatic)
